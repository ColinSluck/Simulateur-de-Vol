# Simulateur de Vol
Project scolaire, creation d'un simulateur de vol/XPlane/OculusRift

### Liste du matériel

-	PC
-	Joystick
-	Oculus rift
-	Carte d’acquisition National Instruments – NI USB-6211
-	Maquette d’avion avec moteurs

### Liste logiciels
-	Qt Creator
-    Ni MAX
-    Wireshark
-    DB Browser for SQLite
